# Auction property images (review)

These are **candidate thumbnails** for the auctions listing cards. They were downloaded from [Pexels](https://www.pexels.com) (free license).

## How to review

1. Open **`preview.html`** in your browser (double-click the file, or drag it into Chrome).
2. Or browse the `.jpg` files in this folder in Finder / Cursor.
3. Check `manifest.json` for labels and which Buy categories each image is meant for.

## Approved for site (24 images)

Live copies are in `public/images/auction-properties/`.

**Excluded** (not houses): `03`, `11`, `15`, `16`–`19`, `31`.

## Re-download

```bash
cd proty-web
node scripts/download-auction-images.mjs
```
